# Sanathan Ethics – Claude Instructions

## Project
Demo astrology website for a client. Currently a **single-file HTML/CSS/JS demo** (`index.html`). No build tools, no framework, no backend.

## Rules
- All code lives in `index.html` — do not create separate files unless explicitly asked.
- Keep JS and CSS embedded in the HTML file.
- Do not add frameworks, libraries, or CDN links unless the user asks.
- Do not refactor or restructure code that wasn't touched by the current task.
- Do not add comments, docstrings, or annotations to unchanged code.

## Color Palette (client-approved)
- `#F5E29E` – Warm pastel gold/yellow

- `#DFCFF3` – Soft lavender
- `#ADE1EF` – Pastel sky blue
- `#B2EAD3` – Mint green
- `#EDBDD5` – Soft pink
- `#2c2055` – Dark accent (brand)
- `#FDFAF3` – Background cream

Always use these colors. Do not introduce new colors without asking.
